[ SERVER REQUIREMENTS ]
=========================================================================

1. PHP Version 5.6.0 or later
2. MySQL Version 5.2.0 or later
3. Curl Support (with SSL)
4. GD2 Image Library
5. PHP JSON support
6. PHP PDO (PDO & MySQL PDO)
7. PHP Memory Limit 64MB (minimum)
8. MySQL Strict Mode disabled
9. Ioncube Loader 10.1.0 or later
